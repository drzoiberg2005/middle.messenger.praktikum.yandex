## Команды

#### Сборка и запуск проекта

```
npm run start
```

#### Запуск в режиме разработки

```
npm run dev
```

#### Запуск сервера

```
npm run server
```

#### Сборка проекта

```
npm run build
```

#### Удаление собранных файлов

```
npm run delete
```

#### Запуск анализатора кода

```
npm run eslint
```

#### Запуск теста
